# wordslearning

Website for memorizing words and definitions. 
Project created to learn Flask framework and improve programming skills

To launch

1. Create .env file  in project  directory
2. Set next values in .env 
`SECRET_KEY=''`
`POSTGRES_URL=''`
`POSTGRES_USER=''`
`POSTGRES_PW=''`
`POSTGRES_DB=''`
3. Run flask db upgrade
